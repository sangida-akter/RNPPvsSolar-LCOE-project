# RNPP-LCOE Project

**Monte Carlo LCOE Simulation for Rooppur Nuclear Power Plant and Solar PV + 4-hour BESS in Bangladesh**

---

## Overview
This repository contains Python scripts, input data, and documentation for deterministic and probabilistic techno-economic analyses of the Rooppur Nuclear Power Plant (RNPP) and utility-scale Solar PV with 4-hour Battery Energy Storage System (BESS) in Bangladesh.

It includes:
- Monte Carlo simulation of LCOE, NPV, and IRR
- Sensitivity analysis using Sobol indices
- Tornado plots for parameter impacts
- Support for policy, financing, and uncertainty assessment
